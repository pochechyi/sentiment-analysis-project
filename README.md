# Review Analysis Platform

Аналитическая платформа для анализа отзывов клиентов.

## Структура проекта

Tech_task1/
├── data/01_raw           # Просто храним Excel файл . Изначально планировалось разбить даты на "сырые данные","очищенные " и "промежуточные"
│   └── data_for_analysis.xlsx
│
├── src/                  # 3 основных файла с кодом
│   ├── loader.py         # Загрузить Excel
│   ├── analyzer.py       # Весь анализ и графики
│   ├── text_analyzer.py  # ТЕКСТОВЫЙ АНАЛИЗ (NLTK)
│   └── utils.py          # Помощники
│
├── reports/              # Сохраянем графики , гистрограммы , отчеты
│
├── main.py               # Запуск всего анализа
├── requirements.txt      # Библиотеки
├── rough_draft/
│   └──exploration.py     # Используем как черновик - Не использовал
└── README.md             # Как пользоваться

## Установка

1. Установить зависимости: `pip install -r requirements.txt`
2. Запустить анализ: `python main.py`
